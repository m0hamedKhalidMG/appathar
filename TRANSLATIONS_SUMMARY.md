## القصيم (Al Qassim)

### معلومات المدينة
- **العربية:** القصيم
- **الإنجليزية:** Al Qassim
- **الوصف:**
  - **AR:** منطقة القصيم الغنية بالآثار والمعالم السياحية
  - **EN:** A region rich in archaeological sites and tourist attractions

### المعالم السياحية (Attractions)

#### 1. شعيب الأدغم / Shuaib Al-Adem
**الوصف:**
- **AR:** يقع شعيب الأدغم شرق منطقة القصيم ويُعد من أهم مواقع الاستيطان البشري القديم في المنطقة. يحتوي الموقع على أدوات حجرية تعود إلى العصور الحجرية المبكرة...
- **EN:** Located east of the Al Qassim region, Shuaib Al-Adem is one of the most important ancient human settlement sites in the area. The site contains stone tools dating back to the early Stone Ages...

#### 2. قصيباء / Qasibaa
**التفصيل:** موطن عنترة بن شداد - مشاهد تاريخية ما قبل الإسلام

#### 3. النقوش والرسوم الصخرية / Rock Inscriptions and Drawings
**التفصيل:** نقوش تمودية ونبطية عتيقة

#### 4-9. معالم إضافية
- درب زبيدة / Darb Zubaidah
- طريق البصرة-مكة / Basra-Mecca Route
- رامة ورامتان / Ramah and Ramatan
- الكتل الحجرية الأثرية / Ancient Stone Blocks
- الرسوم الصخرية القديمة / Ancient Rock Drawings
- مرقب عنيزة / Marqab Aneiza

## ملاحظات مهمة

### 1. الدقة اللغوية
- تمت الترجمة بعناية لضمان الدقة والوضوح
- استخدمت مصطلحات سياحية قياسية

### 2. الاتساق
- جميع الترجمات موحدة عبر جميع المدن
- استخدام نفس المصطلحات للعناصر المتشابهة

### 3. الحفاظ على المعنى
- تم الحفاظ على المعاني والتفاصيل من النسخ الأصلية
- لم يتم حذف أي معلومات أثناء الترجمة

### 4. سهولة الصيانة
- البيانات منظمة بشكل يسهل إضافة أو تعديل الترجمات
- يمكن إضافة اللغات الأخرى بسهولة في المستقبل

## التطبيق العملي

### مثال استخدام في React

```javascript
import citiesData from './data/citiesData';

// الحصول على المعلومات بناءً على اللغة المختارة
function CityCard({ cityKey, language }) {
  const city = citiesData[cityKey];
  const lang = language === 'ar' ? 'ar' : 'en';
  
  return (
    <div>
      <h1>{city[lang].name}</h1>
      <p>{city[lang].description}</p>
      
      {/* عرض الفنادق */}
      {city.hotels.map(hotel => (
        <div key={hotel.image}>
          <h3>{hotel[lang].name}</h3>
          <p>{hotel[lang].description}</p>
          <p>{hotel[lang].address}</p>
        </div>
      ))}
    </div>
  );
}
```
